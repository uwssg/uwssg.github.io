Documentation for the Image Simulations

The initial set of images for the LSST have been generated for the LSST Data Management group (for Data Challenge DC3a). These images and catalogs simulate an r=28 universe as seen through a 15 second LSST exposure. The details of the DC3a data run, the underlying simulations, and additional documentation about the simulations can be found below. The formats for the output catalogs and images can be found here.
  • The requirements document for the LSST DC3a Data Challenge that drove the development of the initial simulated data sets can be found here.
  • Details of the physics underlying the image simulation are available as a short paper prepared for the Dark Energy Task Force
  • A poster on the image simulation framework was presented at the 213th Meeting of the American Astronomical Society
  • More details on the simulation of PSF ellipticities and evaluation of the focal plane design can be found in the following poster presented at the 213th Meeting of the American Astronomical Society
